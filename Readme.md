# 🧠 Mentora - Mental Wellness Platform

![Mentora Logo](assets/images/logo.png)

**Mentora** is a comprehensive AI-powered Mental Wellness, Mentorship, and Medical Consultation Platform specifically designed for the Bangladeshi population. It addresses the growing mental health crisis while providing career guidance and medical support through an integrated ecosystem.

## 🌟 Features

### 🤖 AI Mental Health Chatbot - "মনের বন্ধু"
- 24/7 conversational support in Bengali
- Mood tracking and analysis
- Crisis detection and management
- Weekly mental health reports

### 👨‍🏫 Mentorship System
- Verified mentors for career, academic, and life guidance
- One-on-one and group sessions
- Smart scheduling system
- Progress tracking

### 🏥 Medical Consultation
- Connect with qualified doctors
- Online appointment booking
- Digital prescriptions
- Health records management

### 📚 Learning & Development
- Mental health articles in Bengali
- Video tutorials and courses
- Interactive worksheets
- Success stories

### 👥 Community Features
- Support forums
- Peer support groups
- Virtual events
- Achievement system

### 🎮 Gamification
- Points and badges system
- Daily challenges
- Leaderboards
- Progress rewards

## 🚀 Technology Stack

### Frontend
- HTML5, CSS3, JavaScript
- Bootstrap 5
- jQuery
- Chart.js for visualizations

### Backend
- PHP 7.4+
- MySQL 5.7+
- Apache Server

### Libraries & Tools
- PHPMailer for emails
- DomPDF for PDF generation
- SSLCommerz for payments
- Twilio for SMS
